# FloatBall 浮动球

## 简介
一个简单的Unity2D游戏
通过此项目学习Unity以及Git的使用方法

1. ~~~小球的基本移动~~~
2. ~~~游戏背景素材~~~
3. ~~~小球的发射子弹的攻击逻辑~~~
4. 小球遭受攻击时的逻辑
5. ~~~AI小球的攻击逻辑~~~
6. ~~~AI小球的移动逻辑~~~
7. ~~~AI小球的对象生成逻辑~~~
